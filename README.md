## Installation

On linux based OS,
** Make sure to have the latest version of ruby-dev and gem:**
```
sudo apt-get install ruby-dev
gem update
gem install bundler
```



Quick install

```
bundler install
```

or Manually install Jekyll and its dependencies:

```
gem update
gem install jekyll jekyll-sitemap jekyll-gist
```


Build the static files

```
cd blog;
jekyll build
```
